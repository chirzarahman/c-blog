Java Class Model

Pertama kita harus membuat Class Model. Mengapa ? karena didalam kelas model nya itu sendiri akan mendaftarkan apa aja nanti yang akan digunakan.

Contohnya pada kasus ini kita akan membuat program toko kelontong dimana kita mempunyai Class Model yang terdiri dari 3 member variable yg diisi pada waktu object nantinya diinstansiasi(diisi pada konstruktor). :

  • Nama Barang (String) ==> saya singkat menjadi “nm”
  • Harga Barang (int) ==> saya singkat menjadi “hg”
  • Stok Barang (int) ==> dengan nama tetap “stok”

Setelah itu Klik kanan com.mycompany.tokokelontong pilih “new” pilih lagi “Java Class” seperti gambar berikut.

Buatlah “Java Class” dengan nama “Model” Perlu DIINGAT KAPITALISASI NAMANYA!

di class Model nya codingnya seperti ini

public class Model {

    String nm;
    int hg;
    int stok;

    public Model(String nm, int hg, int stok) {
        this.nm = nm;
        this.hg = hg;
        this.stok = stok;
    }

    public Model() {
    }

    /**
     * @return the nm
     */
    public String getNm() {
        return nm;
    }

    /**
     * @return the hg
     */
    public int getHg() {
        return hg;
    }

    /**
     * @param nm the nm to set
     */
    public void setNm(String nm) {
        this.nm = nm;
    }

    /**
     * @param hg the hg to set
     */
    public void setHg(int hg) {
        this.hg = hg;
    }

    /**
     * @return the stok
     */
    public int getStok() {
        return stok;
    }

    /**
     * @param stok the stok to set
     */
    public void setStok(int stok) {
        this.stok = stok;
    }
}

Method set digunakan untuk mengisi data ke dalam object ArrayList, sedangkan get digunakan untuk menampilkan data

Informasi lengkap : Array

Didalam Class Program marilah kita membuat beberapa Fungsi yang bertujuan efisiensi koding sebab dengan adanya fungsi kita tinggal memanggil fungsi tersebut, misal fungsi “main” manggilnya “main();”.

1. Fungsi menu

kegunaan dari fungsi menu ini adalah menu dimana user ingin input barang, mencari data barang yang sudah diinput, kasir(sebagai perhitungan saat transaksi), ataupun stop(menghentikan program).

Note : Program menu ini menggunakan Switch Case 
//Function Menu
static void menu() throws AWTException {
    Scanner input = new Scanner(System.in);
    shortline();
    System.out.println("\t----- MENU -----");
    shortline();
    System.out.println("1. Input Barang");
    System.out.println("2. Menampilkan Data Barang");
    System.out.println("3. Membeli Barang");
    System.out.println("4. Exit");
    System.out.print("Masukkan Pilihan : ");
    int pilihan = input.nextInt();
    switch (pilihan) {
        case 1: {
            clear();
            input();
        }
        break;
        case 2: {
            clear();
            datasearch();
        }
        break;
        case 3: {
            clear();
            buy();
        }
        break;
        case 4: {
            stop();
        }
    }
}
2. Fungsi input

Kegunaan dari fungsi ini ialah user dapat menginputkan barang sesuai dengan keinginan (didaftarkan pada arraylist), dimana data yang sudah di input dapat dimunculkan saat user memilih menu "Menampilkan Data Barang".

}
3. Fungsi datasearch

didalam fungsi datasearch ini digunakan untuk mempermudah user saat mencari barang yang sudah diinput sebelumnya.

//Function Pencarian Barang
static void datasearch() throws AWTException {
    longline();
    System.out.println("\tSELAMAT DATANG DI TOKO KELONTONG");
    longline();
    Scanner input = new Scanner(System.in).useDelimiter("\n");
    int tandaADA = 0;
    int datake = 0;
    System.out.print("Masukan Nama Barang : ");
    String nm = input.next();
    for (int i = 0; i < lst.size(); i++) {
        if (lst.get(i).getNm().equalsIgnoreCase(nm)) {
            tandaADA = 1;
            datake = i;
        }
    }
    if (tandaADA > 0) {
        Model barang = new Model();
        barang = lst.get(datake);
        System.out.println("Barang yang anda inginkan ada ");
        shortline();
        System.out.println("\tDATA BARANG");
        shortline();
        System.out.println("|" + " Nama Barang     : " + barang.getNm());
        System.out.println("|" + " Harga Satuan    : Rp. " + lst.get(datake).getHg());
        System.out.println("|" + " Stok Barang     : " + lst.get(datake).getStok());
        longline();
        longline();
    } else {
        System.out.println("Barang tidak ditemukan");
        longline();
        longline();
    }
    repeatsearch();
}
4. Fungsi buy

fungsi buy ini seperti program kasir pada umumnya, saat user memasukkan barang yang akan dibeli oleh pelanggan otomatis harga satuan dari barang akan bertambah sesuai jumlah barang yang dibeli dan stok barang akan otomatis berkurang sesuai jumlah barang yang dibeli juga.

Namun, apabila user menginput nama barang yang salah atau tidak ada pada data Arraylist maka fungsi akan mengeluarkan output "Barang tidak ditemukan" dan jika user menginputkan jumlah barang yang tidak sesuai stok barang fungsi akan mengeluarkan output "Stok Barang habis".

//Function Buy
static void buy() throws AWTException {
    longline();
    System.out.println("\tSELAMAT DATANG DI TOKO KELONTONG");
    longline();
    Scanner input = new Scanner(System.in).useDelimiter("\n");
    int tandaADA = 0;
    int datake = 0;
    System.out.print("Masukan Nama Barang : ");
    String nm = input.next();
    for (int i = 0; i < lst.size(); i++) {
        if (lst.get(i).getNm().equalsIgnoreCase(nm)) {
            tandaADA = 1;
            datake = i;
        }
    }
    if (lst.get(datake).getStok() > 0) {
        if (tandaADA > 0) {
            Model barang = new Model();
            barang = lst.get(datake);
            System.out.println("Barang yang anda inginkan ada ");
            shortline();
            System.out.print("Banyak barang yang anda inginkan : ");
            int banyak = input.nextInt();
            if (banyak > lst.get(datake).getStok()) {
                System.out.println("Pembelian Melebihi Stok Barang");
                longline();
                longline();
            } else {
                shortline();
                System.out.println("\tDATA BARANG");
                shortline();
                System.out.println("|" + " Nama Barang     : " + barang.getNm());
                System.out.println("|" + " Harga Satuan    : Rp. " + lst.get(datake).getHg());
                int hn = barang.hg;
                System.out.println("|" + " Total Harga     : Rp. " + (hn * banyak));
                longline();
                longline();
                int stk = lst.get(datake).getStok() - banyak;
                System.out.println("|" + " Sisa Stok       : " + stk);
                barang.setStok(stk);
                longline();
                longline();
            }
        } else {
            System.out.println("Barang tidak ditemukan");
            longline();
            longline();
        }
    } else {
        System.out.println("Stok Barang habis");
        longline();
        longline();
    }

    repeatbuy();
}

fungsi tambahan ini digunakan agar saat program dijalankan mengeluarkan output yang rapi dan enak dilihat.

//Function Stop
static void stop() {
    shortline();
    System.out.println("\tTerima Kasih");
    shortline();
}

//Function Garis Pendek
static void shortline() {
    for (int g = 0; g < 30; g++) {
        System.out.print("=");
    }
    System.out.println("");
}

//Function Garis Panjang
static void longline() {
    for (int g = 0; g < 50; g++) {
        System.out.print("=");
    }
    System.out.println("");
}

//Function clear
static void clear() throws AWTException {
    Robot pressrob = new Robot();
    try {
        pressrob.keyPress(KeyEvent.VK_CONTROL); // press "CTRL"
        pressrob.keyPress(KeyEvent.VK_L); // press "L"
        pressrob.keyRelease(KeyEvent.VK_L); // unpress "L"
        pressrob.keyRelease(KeyEvent.VK_CONTROL); // unpress "CTRL"
        Thread.sleep(1000); // add delay in milisecond, if not there will automatically stop after clear
    } catch (InterruptedException e) {
    }
}
